Get Started in 3 Steps

1

Clone the Repository

git clone https://github.com/yuumuu/Staco.git
cd Staco
2

Start Local Server

# Option 1: npx
npx serve -l 3000 .

# Option 2: Python
python -m http.server 3000
3

Create Your First Page

<layout name="main">
<slot name="content">
<h1>Hello Staco!</h1>
<div x-data="{ count: 0 }">
<button @click="count++">
Clicked: <span x-text="count"></span>
</button>
</div>
</slot>
</layout>
